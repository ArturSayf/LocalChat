# LocalChat

Простой локальный чат на Go с веб-интерфейсом. Сообщения хранятся в памяти сервера (последние 10 сообщений).

## Возможности

- Отправка сообщений с указанием имени пользователя
- Автоматическое обновление списка сообщений каждую секунду
- Хранение только последних 10 сообщений
- Простой и понятный интерфейс

## Технологии

- **Backend**: Go (стандартная библиотека `net/http`)
- **Frontend**: HTML, JavaScript (Fetch API)
- **Форматы данных**: JSON, multipart/form-data

## Установка и запуск

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/yourusername/localchat.git
   cd localchat
Убедитесь, что установлен Go (версия 1.25.1 или выше)

bash
go version
Запустите сервер

bash
go run main.go
Откройте браузер и перейдите по адресу:

text
http://localhost:8090
Использование
Введите имя пользователя в поле "Username"

Напишите сообщение в поле "Message"

Нажмите кнопку "Send" для отправки

Сообщения автоматически обновляются каждую секунду

Структура проекта
main.go - основной файл сервера на Go

index.html - HTML-шаблон с JavaScript-логикой (встроен в бинарник через //go:embed)

go.mod - файл модуля Go

API Endpoints
GET / - главная страница чата

POST /send-message - отправка нового сообщения (multipart/form-data)

GET /get-messages - получение всех сообщений (JSON)

Формат сообщений
json
{
  "id": 1,
  "name": "Имя пользователя",
  "message": "Текст сообщения"
}
Особенности
Сообщения хранятся только в памяти и теряются при перезапуске сервера

Одновременно хранится не более 10 последних сообщений

Для корректной работы требуется включенный JavaScript в браузере

Требования
Go 1.25.1 или выше

Современный веб-браузер с поддержкой JavaScript

Порт 8090 должен быть свободен
